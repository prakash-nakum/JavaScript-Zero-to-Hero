# JavaScript-Zero-to-Hero
Anime web project built with HTML, Tailwind CSS v3, JavaScript, PHP, and MySQL. The frontend UI uses Tailwind CDN with simple page navigation, while the backend provides PHP APIs connected to a MySQL database. Frontend is ready for deployment on Vercel with a separate PHP backend server.

# Anime Web Project (HTML, Tailwind CSS, JS, PHP, MySQL)

This project is a simple anime-based web application built using **HTML, Tailwind CSS v3, JavaScript, PHP, and MySQL**.

The frontend interface is designed using **Tailwind CSS CDN**, while the backend APIs are built with **PHP** and connected to a **MySQL database** for dynamic data handling.

## Features

* Responsive UI using Tailwind CSS
* Simple navigation between pages
* Example page routing
* PHP API integration
* MySQL database connection
* Ready for frontend deployment on Vercel

## Project Structure

Frontend:

* HTML pages
* Tailwind CSS styling
* JavaScript for dynamic behavior

Backend:

* PHP APIs
* MySQL database operations

## Deployment

* Frontend hosted on Vercel
* Backend hosted on PHP hosting server

## Purpose

This project is mainly created for learning how to integrate:

* Frontend (HTML, Tailwind, JS)
* Backend (PHP APIs)
* Database (MySQL)
* Deployment workflow
